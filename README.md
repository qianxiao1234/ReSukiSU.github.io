# ReSukiSU-Docs
 The ReSukiSU's Document Website

## Devlopment

```shell
npm install -g yarn@1.22.22
yarn install

# dev
yarn docs:dev --host

# build
yarn docs:build

# preview
yarn docs:preview
```

## Notes

This Website using `Misans VF`(source included) and `JetBrains Mono`(via Zstatic CDN) fonts.
